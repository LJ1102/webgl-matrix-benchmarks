This page benchmarks the performance of several matrix libraries intended for use with WebGL: glMatrix, mjs, CanvasMatrix, EWGL_math, and tdl.
Benchmark times are averaged over 10 runs of 20,000 iterations of the target operation.

Adapted from the benchmarks Brandon Jones created in his glmatrix library.
JavaScript Plotting library: Flotr; Sortable Table by Andrew Tetlaw.
The source code for these benchmarks can be found here on github.
